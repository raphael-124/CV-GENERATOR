# CV-GENERATOR

Simple CV/resume generator built with Python.

Contents
- `app.py` — Flask app entrypoint.
- `templates/index.html` — HTML template used to render the CV.
- `static/style.css` — Basic styling for the generated CV.
- `utils/generator.py` — Helper functions to create CV content.

Quick start

1. Create and activate a Python environment (optional but recommended):

```powershell
python -m venv .venv
.\\.venv\\Scripts\\Activate.ps1
```

2. Install dependencies:

```powershell
pip install -r requirements.txt
```

3. Run the app:

```powershell
python app.py
```

Open `http://127.0.0.1:5000` in your browser.
